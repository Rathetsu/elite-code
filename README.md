# EliteCode

EliteCode is a coding challenge platform inspired by LeetCode, providing a space for users to practice coding and algorithm problems.

## Technologies Used

- **Frontend**: React, NextJS, TypeScript, Tailwind CSS, RecoilJS
- **Backend**: Firebase
